# 在线租车网站 (ZuChe Online Car Rental)

## 项目背景

本项目是一款面向大众的在线租车服务平台，旨在为用户提供便捷、高效、透明的租车服务体验。随着共享经济的快速发展和人们对出行灵活性的需求增加，租车服务已成为现代城市出行的重要组成部分。本系统结合人工智能技术，提供智能推荐、智能客服等创新功能，提升用户租车体验。

**参赛类别**：软件应用与开发 - Web 应用开发

## 功能说明

### 用户端功能
- **用户认证**：注册、登录、密码修改、个人信息管理
- **车辆浏览**：分类筛选、品牌筛选、价格排序、热门排序、车辆详情查看
- **车辆搜索**：按时间、地点、车型多维度搜索可用车辆
- **订单管理**：创建订单、订单支付、订单取消、订单评价、订单历史查询
- **个人中心**：收藏管理、订单管理、个人信息修改
- **智能推荐**：基于用户行为和偏好的个性化车辆推荐
- **AI智能客服**：基于知识库的智能问答，解答租车相关问题

### 管理端功能
- **数据看板**：订单统计、收入分析、用户增长等数据可视化
- **用户管理**：用户列表查询、用户状态管理
- **车辆管理**：车辆上架、下架、编辑、分类管理、品牌管理
- **门店管理**：门店信息维护、门店运营状态管理
- **订单管理**：订单列表、订单状态流转、订单详情查看
- **知识库管理**：FAQ 内容维护、智能客服知识库配置
- **数据导出**：订单、用户、车辆等数据 Excel 导出

## 使用方法

### 环境要求
- JDK 8+
- Node.js 16+
- MySQL 8.0+
- Redis 6.0+

### 后端启动

1. 创建数据库
```sql
CREATE DATABASE zuche DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE zuche;
SOURCE zuche.sql;
```

2. 修改配置文件 `zuche-server/src/main/resources/application.yml` 中的数据库连接信息

3. 启动后端服务
```bash
cd zuche-server
mvn spring-boot:run
```

后端服务将在 `http://localhost:8080/api` 启动

### 前端启动

1. 安装依赖
```bash
cd zuche-web
npm install
```

2. 启动开发服务器
```bash
npm run dev
```

前端服务将在 `http://localhost:5173` 启动

## 核心功能模块

### 1. 用户认证模块
基于 Spring Security 和 JWT 实现安全的用户认证机制，支持登录状态保持、Token 刷新、密码加密等功能。

### 2. 车辆浏览模块
支持多维度筛选、排序、分页，提供车辆详情查看、车辆图片画廊、用户评价展示等功能。

### 3. 订单流程模块
完整的租车订单流程：选择车辆 → 填写信息 → 提交订单 → 支付 → 取车 → 还车 → 评价，支持订单状态实时跟踪。

### 4. 智能推荐模块
基于用户行为数据和车辆相似度算法，实现个性化车辆推荐，提升用户租车体验。

### 5. AI智能客服模块
基于 RAG（检索增强生成）技术，结合租车领域知识库，提供智能问答服务。

### 6. 数据可视化模块
使用 ECharts 实现订单统计、收入分析、用户增长等数据看板，辅助运营决策。
