# 租车网站系统安装说明

## 一、环境要求

### 后端环境
- JDK 8+
- Maven 3.6+
- MySQL 8.0+
- Redis 6.0+

### 前端环境
- Node.js 16.18+
- npm 8.0+

### 硬件配置
- 处理器：Intel i5 / AMD Ryzen 5+
- 内存：8GB+
- 存储：20GB+

---

## 二、数据库配置

1. **创建数据库**
```sql
CREATE DATABASE zuche DEFAULT CHARACTER SET utf8mb4;
```

2. **修改配置** (`zuche-server/src/main/resources/application.yml`)
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/zuche?serverTimezone=Asia/Shanghai
    username: root
    password: 1234  # 修改为你的密码
  redis:
    host: localhost
    port: 6379
```

---

## 三、后端启动

```bash
cd zuche-server
mvn spring-boot:run
```

- 服务地址：`http://localhost:8080/api`

---

## 四、前端启动

```bash
cd zuche-web
npm install  # 首次运行
npm run dev
```

- 访问地址：`http://localhost:3000`
- 管理后台：`http://localhost:3000/admin`

---

## 五、验证服务

1. 后端启动成功提示：`Started ZucheServerApplication`
2. 前端启动成功提示：`VITE ready in XXX ms`
3. 浏览器访问 `http://localhost:3000` 显示首页

---

## 六、常见问题

| 问题 | 解决方法 |
|-----|---------|
| 端口8080被占用 | 修改 `application.yml` 端口或停止占用进程 |
| 数据库连接失败 | 确认MySQL启动，检查用户名密码 |
| Redis连接失败 | 确认Redis启动，检查配置 |
| 前端依赖安装失败 | `npm cache clean --force && rm -rf node_modules && npm install` |

---

| 配置项 | 默认值 | 修改位置 |
|-------|-------|---------|
| 后端端口 | 8080 | `application.yml` |
