# 租车系统 - 订单逻辑重构验证清单

## 后端验证
- [ ] 订单创建后状态是否为未付款
- [ ] 支付成功后订单状态是否变更为待取车
- [ ] 门店管理员确认用车后订单状态是否变更为用车中
- [ ] 门店管理员确认还车后订单状态是否变更为已还车
- [ ] 30分钟未付款订单是否自动取消
- [ ] 订单状态变更的API是否正常工作

## 前端验证
- [ ] 点击预定按钮是否弹出支付确认弹窗
- [ ] 支付确认弹窗是否显示订单详情
- [ ] 支付确认弹窗是否显示30分钟倒计时
- [ ] 点击确认支付后订单状态是否变更为待取车
- [ ] 订单页面是否只显示全部、待取车、用车中、已还车四个页签
- [ ] 页签过滤功能是否正常工作
- [ ] 门店管理员是否可以看到确认用车和还车按钮
- [ ] 点击确认按钮后订单状态是否正确变更

## 整体流程验证
- [ ] 完整订单流程是否顺畅
- [ ] 30分钟自动取消功能是否正常工作
- [ ] 所有功能是否符合需求文档要求
- [ ] 系统是否稳定运行
